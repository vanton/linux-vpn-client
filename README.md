# linux vpn client

## 项目介绍

linux 下连接 l2tp vpn

## 结构说明

vpn_client
    |-  install_vpn.sh  安装脚本。仅做了 centOS 和 ubuntu 配置。
    |-  start_vpn.sh  启动脚本
    |-  stop_vpn.sh  停止脚本

## 使用说明

注意：普通用户需要 sudo 运行。
